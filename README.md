# 🎮 Mining Bot - Telegram AFK Mining Game

AFK-минигрант по добыче руды для Telegram с механиками прогрессии, боссами и экономикой.

## 📋 Функции

- ⛏️ **AFK-добыча руды** - запускайте добычу и возвращайтесь позже
- 🏔️ **8 шахт** - открываются каждые 2 уровня
- 🎆 **Плазма** - редкий ресурс для улучшений
- 📦 **Кейсы** - деревянные, железные, золотые, легендарные
- 🎒 **Инвентарь** - руда, материалы, расходники
- 💵 **Экономика** - продажа руды, переводы между игроками
- 📈 **Уровни** - прогрессия с открытием нового контента
- 👹 **Боссы** - каждые 15 уровней
- ⏳ **Flood Wait** - защита от слишком частых обновлений

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone <repository_url>
cd mining_bot
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

Отредактируйте `.env`:
- `BOT_TOKEN` - токен от @BotFather
- `DATABASE_URL` - URL базы данных (SQLite по умолчанию)

### 5. Запуск бота

```bash
python bot.py
```

## 🎮 Геймплей

### Начало игры

1. Запустите бота командой `/start`
2. Откройте меню добычи (⛏️ Добывать)
3. Выберите шахту (доступна "Земля I")
4. Нажмите "🔨 Добывать"

### Добыча ресурсов

- Запустите добычу на 5 минут
- Возвращайтесь каждые 2-3 минуты для обновления (🔄)
- Собирайте ресурсы после завершения
- Руда автоматически добавляется в инвентарь

### Экономика

- **Продажа руды**: `/inv` → "Продать всю руду"
- **Переводы**: `перевести @username сумма`
- **Улучшения**: магазин за деньги

### Прогрессия

- **Каждые 2 уровня**: новая шахта
- **Каждые 15 уровней**: босс
- **Лимит переводов**: растёт с уровнем

## 🛠️ Технологии

- **Python 3.8+**
- **aiogram 3.x** - асинхронный фреймворк для Telegram Bot API
- **SQLAlchemy 2.x** - ORM для работы с БД
- **SQLite/PostgreSQL** - база данных
- **FSM** - машина состояний для AFK-режима

## ⚙️ Конфигурация

В `config.py` можно изменить:

- `BASE_MINING_TIME` - время добычи (секунды)
- `FLOOD_WAIT` - задержка между обновлениями
- `PLASMA_CHANCE` - шанс выпадения плазмы
- `CASE_CHANCE` - шанс выпадения кейса
- Цены и шансы руды

## 🐛 Решение проблем

### Бот не запускается
- Проверьте `BOT_TOKEN` в `.env`
- Убедитесь, что все зависимости установлены

### Ошибки базы данных
- Удалите файл `mining_bot.db` для сброса SQLite
- Проверьте подключение PostgreSQL

### Flood Wait
- Подождите 2-3 минуты между обновлениями добычи
