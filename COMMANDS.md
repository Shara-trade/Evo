# 📋 Команды Mining Bot

## Основные команды

| Команда | Описание | Пример |
|---------|----------|--------|
| `/start` | Запуск бота, приветствие | `/start` |
| `/menu` | Главное меню | `/menu` |
| `/bal` | Проверка баланса | `/bal` или `Б` |
| `/inv` | Инвентарь игрока | `/inv` |
| `/lvl` | Информация об уровне | `/lvl` |
| `/limit` | Лимит на переводы | `/limit` или `Лимит` |

## Inline-кнопки меню

### Главное меню
- ⛏️ **Добывать** - меню добычи руды
- 🎒 **Инвентарь** - просмотр ресурсов
- 💵 **Баланс** - деньги и плазма
- 📈 **Уровень** - прогресс и мощность
- 🔄 **Перевод** - перевод денег
- 🏪 **Магазин** - улучшения
- 👥 **Профиль** - статистика игрока

### Меню добычи (не активно)
- 🔨 **Добывать** - начать добычу
- Закрыть - закрыть меню

### Меню добычи (активно)
- 🚫 **Остановить** - завершить добычу
- Закрыть - закрыть меню
- 🔄 - обновить статистику

### Меню добычи (завершено)
- 🧱 **Собрать** - собрать ресурсы
- Закрыть - закрыть меню

### Выбор шахты
- Список доступных шахт с уровнем требования
- 🔙 **Назад** - вернуться к добыче

### Инвентарь
- 💵 **Продать всю руду** - конвертация в деньги
- 🔙 **Назад** - в главное меню

### Магазин
- ⚒️ **Улучшить кирку** - +×0.5 мощности
- 🚀 **Купить бустер** - +×1.0 мощности
- 🔙 **Назад** - в главное меню

## Текстовые команды

### Перевод денег
```
перевести @username сумма
```
Пример: `перевести @friend 500`

### Сокращения
- `Б` - аналог `/bal` (баланс)
- `Лимит` - аналог `/limit`

## Механики

### Добыча руды
1. Откройте меню добычи (⛏️ Добывать)
2. Выберите шахту (если доступно несколько)
3. Нажмите 🔨 **Добывать**
4. Вернитесь через 2-3 минуты для обновления (🔄)
5. После завершения нажмите 🧱 **Собрать**

### Продажа руды
1. Откройте инвентарь (`/inv`)
2. Нажмите 💵 **Продать всю руду**
3. Руда конвертируется в деньги по ценам из config.py

### Улучшения
- **Кирка**: увеличивает базовую мощность
- **Бустер**: множитель мощности
- Общая мощность = кирка × бустер

### Уровень и прогрессия
- Опыт получают за добычу руды
- Каждый уровень увеличивает лимит переводов
- Новые шахты открываются каждые 2 уровня
- Боссы появляются каждые 15 уровней

### Ресурсы

#### Руда
- Выпадает при добыче
- Продаётся за деньги
- Цена зависит от редкости

#### Плазма (🎆)
- Редкий ресурс (5% шанс)
- Нужна для улучшений
- Не продаётся

#### Кейсы (📦)
- Выпадают редко (2% шанс)
- Содержат плазму
- Типы: деревянный, железный, золотой, легендарный

### Ограничения

#### Flood Wait
- Обновлять добычу можно раз в 2 минуты
- Иначе временная блокировка

#### Лимит переводов
- База: 1000💰
- +100💰 за каждый уровень
- Сбрасывается ежедневно

#### Время добычи
- Максимум: 5 минут
- После нужно собрать ресурсы

## Советы новичкам

1. 🎯 **Начните с Земли I** - не спешите открывать новые шахты
2. 💰 **Копите на улучшения** - мощность важна для прогресса
3. 🎆 **Берегите плазму** - она важнее денег
4. ⏰ **Проверяйте бота каждые 2-3 минуты** - для обновления добычи
5. 📦 **Открывайте кейсы** - там ценная плазма
6. 🆙 **Повышайте уровень** - открываются новые шахты
7. 💸 **Следите за лимитом** - для переводов друзьям

## Частые вопросы

**Q: Как увеличить время добычи?**
A: Время фиксированное (5 минут). После нужно собрать ресурсы и начать заново.

**Q: Где взять плазму?**
A: Плазма выпадает случайно при добыче (5% шанс) или из кейсов.

**Q: Как открыть новую шахту?**
A: Повышайте уровень. Каждая шахта требует определённый уровень.

**Q: Что делают боссы?**
A: Боссы появляются каждые 15 уровней. Механика боя в разработке.

**Q: Можно ли потерять ресурсы?**
A: Нет, ресурсы сохраняются в инвентаре до продажи или использования.
