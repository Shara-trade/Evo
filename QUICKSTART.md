# 🚀 Быстрый старт Mining Bot

## 1. Установка (3 минуты)

### Windows
```bash
# Запустите скрипт
run.bat
```

### Linux/Mac
```bash
chmod +x run.sh
./run.sh
```

## 2. Настройка (1 минута)

1. Откройте `.env` файл
2. Вставьте токен от @BotFather
3. Сохраните файл

## 3. Запуск

```bash
python bot.py
```

## 4. Проверка

1. Найдите бота в Telegram
2. Нажмите `/start`
3. Проверьте меню

## 📁 Структура проекта

```
mining_bot/
├── bot.py              # Главный файл (запускать это)
├── config.py           # Настройки игры
├── database.py         # База данных
├── handlers.py         # Команды и кнопки
├── keyboards.py        # Клавиатуры
├── states.py           # FSM состояния
├── middlewares.py      # Middleware БД
├── requirements.txt    # Зависимости
├── .env                # Ваши настройки (создаётся вручную)
└── README.md           # Полная документация
```

## 🎮 Как играть

1. **Начать добычу**: ⛏️ Добывать → 🔨 Добывать
2. **Собрать ресурсы**: Вернитесь через 5 мин → 🧱 Собрать
3. **Продать руду**: /inv → Продать всю руду
4. **Улучшиться**: 🏪 Магазин → Улучшить кирку

## ⚙️ Настройки

В `config.py` можно изменить:
- Время добычи: `BASE_MINING_TIME = 300` (5 минут)
- Flood Wait: `FLOOD_WAIT = 120` (2 минуты)
- Шансы: `PLASMA_CHANCE = 5` (5%)

## 🐛 Проблемы?

- **Бот не запускается**: Проверьте токен в `.env`
- **Ошибки БД**: Удалите `mining_bot.db`
- **Зависимости**: `pip install -r requirements.txt`
